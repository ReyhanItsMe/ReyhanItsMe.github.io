Cara Membeli Produk

Panduan lengkap untuk membeli produk digital di DEVACODE

Pilih Produk

Jelajahi katalog produk kami dan pilih produk yang Anda inginkan. Anda bisa menggunakan filter untuk mempermudah pencarian.

Lihat Detail Produk

Klik produk untuk melihat detail lengkap termasuk deskripsi, fitur, preview, dan harga.

Klik "Beli Sekarang"

Setelah yakin, klik tombol "Beli Sekarang" untuk melanjutkan ke proses checkout.

Review Pesanan

Periksa kembali detail pesanan Anda di popup struk pembayaran sebelum melanjutkan.

Lanjutkan ke WhatsApp

Klik tombol untuk melanjutkan ke WhatsApp. Admin akan membantu proses pembayaran Anda.

Lakukan Pembayaran

Transfer sesuai nominal yang diberikan ke rekening/e-wallet yang tersedia.

Terima Produk

Setelah pembayaran dikonfirmasi, produk akan dikirim ke email/WhatsApp Anda dalam hitungan menit.

Tips: Simpan bukti pembayaran dan chat dengan admin untuk referensi jika diperlukan di kemudian hari.

Cara Download Produk

Panduan mengakses produk yang sudah dibeli

Via Email

  1. Cek inbox email yang Anda berikan saat order
  2. Cari email dari DEVACODE dengan subjek "Produk Anda Siap"
  3. Klik link download yang tersedia di email
  4. Jika tidak ada di inbox, cek folder Spam/Junk

Via WhatsApp

  1. Admin akan mengirim file langsung ke chat WhatsApp Anda
  2. Download file dengan tap & hold lalu pilih "Save"
  3. Untuk file besar, link Google Drive/MediaFire akan diberikan

Via Cloud Storage

  1. Link Google Drive/Dropbox akan diberikan
  2. Klik link untuk membuka halaman download
  3. Klik tombol Download atau ikon panah ke bawah
  4. Extract file ZIP menggunakan aplikasi seperti WinRAR atau 7-Zip

Penting: Link download memiliki masa berlaku. Segera download produk Anda setelah menerima link.

Penggunaan Produk

Cara menggunakan berbagai jenis produk digital

Web HTML/Template

  1. Extract file ZIP
  2. Buka dengan code editor (VS Code, dll)
  3. Edit konten sesuai kebutuhan
  4. Preview dengan browser
  5. Upload ke hosting

Preset Alight Motion

  1. Download file .alight
  2. Buka aplikasi Alight Motion
  3. Tap titik 3 → Import
  4. Pilih file preset
  5. Edit dan export video

Template PLP/Pixellab

  1. Download file .plp
  2. Buka aplikasi Pixellab
  3. Menu → Load Project
  4. Pilih file template
  5. Edit teks dan warna

Komponen HTML

  1. Buka file komponen
  2. Copy kode yang dibutuhkan
  3. Paste ke project Anda
  4. Sesuaikan styling
  5. Integrasikan dengan website

Setiap produk dilengkapi dengan dokumentasi/README yang menjelaskan cara penggunaan secara detail.

Peringatan & Larangan

Hal-hal yang tidak diperbolehkan

Dilarang Keras

  • Menjual ulang (Resell) - Menjual produk dalam bentuk asli atau sedikit modifikasi
  • Redistribusi gratis - Membagikan file ke orang lain secara gratis
  • Upload ke platform sharing - Upload ke situs download gratis
  • Klaim sebagai karya sendiri - Mengakui sebagai kreator asli
  • Penggunaan ilegal - Untuk konten melanggar hukum

Konsekuensi Pelanggaran

  • 1. Pembatalan lisensi tanpa refund
  • 2. Blacklist dari pembelian di masa depan
  • 3. DMCA takedown untuk konten yang melanggar
  • 4. Tuntutan hukum untuk pelanggaran berat

Dengan membeli produk, Anda dianggap telah membaca dan menyetujui semua ketentuan lisensi.

Kebijakan Refund

Ketentuan pengembalian dana

Catatan Penting: Karena sifat produk digital yang dapat langsung diduplikasi setelah diterima, kebijakan refund kami bersifat ketat.

Refund Diterima

  • Produk tidak sesuai deskripsi
  • File corrupt/rusak dan tidak bisa diperbaiki
  • Kesalahan teknis dari pihak kami
  • Double payment (bayar 2x)

Refund Ditolak

  • Berubah pikiran setelah download
  • Tidak punya skill untuk menggunakan
  • Sudah memodifikasi produk
  • Pengajuan lebih dari 7 hari

Prosedur Pengajuan Refund

  1. Hubungi admin via WhatsApp dalam waktu 7 hari setelah pembelian
  2. Jelaskan alasan refund dengan detail
  3. Sertakan bukti (screenshot error, dll)
  4. Tunggu proses review (1-3 hari kerja)
  5. Jika disetujui, dana dikembalikan ke metode pembayaran asal
